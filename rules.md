# 纳斯达克 70% 策略 v3.0 速查

## 核心比例

- 总资产 = 纳指持仓 + 备用金 + 其他现金
- 纳指持仓目标：40%
- 纳指备用金目标：30%
- 其他现金目标：30%
- 纳指相关总敞口上限：70%（持仓 + 备用金）

## 月度现金流

- 每月发工资后操作 1 次，默认 ¥4,000。
- 持仓 < 40% 且备用金 < 30%：¥1,600 买持仓，¥1,200 补备用金，¥1,200 进其他现金。
- 持仓 < 40% 且备用金 >= 30%：¥1,600 买持仓，¥2,400 进其他现金。
- 持仓 >= 40%：¥4,000 全进其他现金。

## 回撤分档

- 判定基准：NDX，按最近 1 年高点计算回撤。
- 首次触发 -10% 时，锁定当时备用金余额作为本轮快照。
- 本轮 -10% / -15% / -20% / -25% 均按快照金额计算。
- 档位比例：20% / 30% / 30% / 20%。
- 每档只触发一次，不重复、不回补、不提前动用后续档位。

## 冻结条件

任一满足即冻结：

- 持仓占比 >= 40%
- 纳指相关总敞口 >= 70%
- 连续 3 周上涨且出现怕踏空情绪

冻结后停止主动买入，月度现金流全进其他现金。

## 复核

- 每 3 个月复核一次。
- 允许重算总资产、40%/30% 金额基准、DCA 回补节奏。
- 禁止因短期涨跌修改比例、回撤档位或核心框架。
